## 文字版 vs 卡拉OK 的区别

| | 文字版 | 卡拉OK（动感） |
|---|---|---|
| 渐变 | ❌ | ✅ 上/下两色 |
| 描边 | ❌ | ✅ |
| 进度染色 | ❌ 不染 | ✅ 逐字染色 |
| 颜色 | 单色（已/未播放同色或两色） | 上半已播放 / 下半未播放 |
| 适合 | 简洁、阅读为主 | 演出效果、K歌 |
