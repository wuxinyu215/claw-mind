## 教训 / lessons

- **配置前先 backup**（`openclaw.json.bak.pre-*-YYYYMMDD-HHMMSS` 命名规范）
- **base64 编码处理 secret 进命令** — 不是真加密，是回避 grep/命令历史，明说给用户
- **PowerShell ScriptBlock 解析坑** — `{` 开头字符串会被解析，绕开方式：here-string + 文件 + `--batch-file`
- **OpenClaw dynamic model discovery 强大** — provider 端点列出什么模型 OpenClaw 就能用，不需要手动加。但**仍需手动加到 allowlist** 才能 `/model` 选到
- **OpenClaw SecretRef 解析时序** — `${VAR}` 是 SecretRef 不是字符串替换，gateway 启动时缓存 env，User 级 env var 重启后生效，**写 .env 是 hot-reload 友好的保险**
- **不要假设 LLM 模型版本** — V3/V4 这种命名变更快，先看 dynamic discovery 别凭印象
