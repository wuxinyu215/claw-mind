## 一个建议

以后如果想**保留 lock**（比如 commit 完还要继续改）：
```bash
svn commit --no-unlock -m "..."  # commit 但保留锁
```

如果想**释放 lock**（commit 完让别人也能改）：
```bash
svn commit -m "..."  # 默认行为
```
